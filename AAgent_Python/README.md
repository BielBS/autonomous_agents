# Running instructions:

1. Open the environment
2. To test the Alone Scenario
    1. Pick one of the first 2 scenes
        1. Run 'AAgent_BT.py AAgent-1.json'  
        2. Press start and enjoy
    2. For the AAC scene
        1. Run 'AAgent_BT.py AAgent-Alpha.json' 
        2. Press start and enjoy
3. To test the Critters Scenario
    1. Select the AAC scene
        1. Run 'Spawner.py APackAstroCrittersIgnore.json'
        2. Press start and enjoy
4. To test the Collect and Run Scenario
    1. Select the AAC scene
        1. Run 'Spawner.py APackAstroCrittersAvoid.json'
        2. Press start and enjoy
